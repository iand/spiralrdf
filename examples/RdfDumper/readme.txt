RdfDumper - a simple utility to dump some RDF to the console

Usage:  

RdfDumper.exe http://example.com/rdf

Requires the following to be in the same directory:
SemPlan.RdfLib.dll
SemPlan.RdfLib.XsltParser.dll
RdfXmlToTriples.xsl
RdfXmlValidator.xsl
