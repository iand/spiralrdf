Shootout - a port of Simile project's shootout performance test

Usage:  

Shootout.exe <type> <number of triples>

<type> can be one of mem or db to use either MemoryTripleStore
or DatabaseTripleStore


Requires the following to be in the same directory:
SemPlan.Spiral.dll
SemPlan.Spiral.MySql.dll
Shootout.exe.config
